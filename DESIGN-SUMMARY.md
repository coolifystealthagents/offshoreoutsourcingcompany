# Design summary

- Design concept: Global logistics
- Layout personality: wide layered routes
- Typography: Archivo Black headings with Archivo body text
- Color palette: primary #111827, secondary #facc15, accent #06b6d4, background #f8fafc, text #111827, muted #64748b
- Shape language: Mixed geometric shapes
- Visual motif: route lines and port labels
- Unique layout choices: first sections use Route planner, then a custom service index, process ribbon, question block, and resource strip with varied widths and alignments.
- Difference from previous sites: this redesign changes the hero composition, section ordering, image treatment, card style, spacing rhythm, button shape, and footer treatment for this domain.
